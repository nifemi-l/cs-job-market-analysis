# EECS 767 Project — CS Job-Market Sentiment

Sentiment analysis of CS career subreddits across pre-COVID, COVID, and post-COVID eras.

## Setup

```bash
pip install -r requirements.txt
```

## Data

The Sentiment140 training data is too large for GitHub, so you need to download it yourself:

1. Go to https://www.kaggle.com/datasets/kazanova/sentiment140/data
2. Download and unzip the archive
3. Place `training.1600000.processed.noemoticon.csv` in `src/data/sentiment140/`

## Project structure

```
docs/                   class info + proposal
src/
  data/
    sentiment140/       Sentiment140 CSV goes here (gitignored)
  models/               trained vectorizer + classifiers (generated by train script)
  preprocessing.py      text cleaning and data loading (no sklearn)
  train_sentiment.py    trains logistic regression + naive bayes on Sentiment140
requirements.txt
```

## Running the training

```bash
cd src
python3 train_sentiment.py
```

This preprocesses the Sentiment140 data, does an 80/20 train/test split, fits TF-IDF, and trains both models. Results and saved `.joblib` files go into `src/models/`.
